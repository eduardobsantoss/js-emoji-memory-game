# 🎮 Jogo da Memória com Emojis

Um divertido jogo da memória desenvolvido em **HTML**, **CSS** e **JavaScript** com um tema vibrante e elementos visuais de emojis! Teste sua memória combinando pares de cartas enquanto desfruta de um design colorido e interativo.

---

## 🚀 Funcionalidades

- **Interface Colorida**: Design vibrante com um gradiente de roxo e rosa.
- **Sistema de Vitória**: Uma mensagem de "Você venceu!" aparece ao finalizar o jogo.
- **Botão Reset**: Permite reiniciar o jogo para começar novamente.
- **Elementos Interativos**: Emojis para tornar o jogo visualmente atraente.

---

## 🎯 Objetivo

Combine os pares de emojis o mais rápido possível para completar o jogo e ver a mensagem de vitória!

---

## 💻 Tecnologias Utilizadas

- **HTML5**: Estrutura do jogo.
- **CSS3**: Estilização com gradientes e layout responsivo.
- **JavaScript**: Lógica do jogo, manipulação de eventos e verificação de vitória.

---

## 🎮 Como Jogar

##### 1. Clique em duas cartas para revelá-las.
##### 2. Se as cartas forem iguais, elas permanecerão viradas.
##### 3. Se não forem iguais, as cartas serão escondidas novamente.
##### 4. Continue jogando até encontrar todos os pares.
##### 5. Ao finalizar, uma mensagem de vitória será exibida.

---

## 🚀 Como Executar

##### 1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/jogo-da-memoria.git
   ```

##### 2. Navegue até o diretório do projeto:
```cd jogo-da-memoria```

---

## 🧪 Exemplos de Teste
- Situação Inicial: Todas as cartas estão viradas para baixo.
- Interação: Clique em duas cartas para verificar se formam um par.
- Vitória: Todas as cartas combinadas resultam na exibição da mensagem "Você venceu!".
